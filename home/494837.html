Controle de Expedição - Confisa Sistemas

Neste artigo veremos como utilizar o módulo FAT0040 - Controle de Expedição.


O Controle de Expedição serve para controlar a entrega das faturas. Acesse o módulo através do menu <Faturamento\Expedição/Romaneios\Controle de Expedição> ou pelas teclas de atalho 'CTRL + F', pesquisando pelo código 'FAT0040' ou pelo termo 'expedição':

Figura 1: Acessando o módulo Controle de Expedição. 


Selecione um período de faturas emitidas e clique no botão 'Buscar': 

Figura 2: Filtrando faturas de um período


Depois selecione uma ou mais faturas para marcá-las como 'Em transporte', indicando que o pedido saiu para entrega. Da mesma maneira, clique no botão 'Finalizar' para marcar o pedido como 'Entregue'. Observe como o status das faturas se altera.

É possível fazer o acompanhamento das faturas usando o Relatório de Expedição. Para mais detalhes, clique aqui.
