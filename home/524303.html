Substituir data de entrega - Confisa Sistemas

Neste artigo veremos como configurar o sistema para substituir a data de entrega que está na fatura para a data atual.


Quando montamos um pedido de venda, podemos definir uma data de entrega prevista que foi combinada com o cliente. Ao transformar esse pedido em fatura, o sistema fará a baixa de estoque na data de entrega informada, o que pode fazer com que o estoque seja baixado no futuro caso a data não seja alterada para o dia de faturamento.

Figura 1: Informar data de entrega no pedido.


Para evitar esse problema, podemos ativar a configuração "Substituir data de entrega" na filial, a qual troca data de entrega informada no pedido pela data atual, fazendo a baixa de estoque no momento em que o faturamento foi feito.Figura 2: Ativar a configuração "Substituir data de entrega".


Ao salvar a fatura, o sistema coloca a data de entrega para o dia de hoje automaticamente, baixando o estoque e gerando as contas a receber com data de emissão na mesma data do faturamento.
