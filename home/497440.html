Fatura para cliente inativo - Confisa Sistemas

Neste artigo veremos como configurar o sistema para emitir avisos caso tente-se emitir faturas para clientes inativos (Que não compram há algum tempo).


Quando um cliente deixa de comprar por um certo tempo, o sistema automaticamente o inativará. É possível configurar o sistema para emitir avisos caso tente-se fazer vendas para ele, podendo o usuário continuar a venda ou não.

Para isso, primeiramente devemos deixar a configuração 'Bloquear fatura se o cliente estiver inativo' como 'Não'.

Figura 1: Configuração 'Bloquear fatura se o cliente estiver inativo'.


Ao tentar faturar para um cliente inativo, caso a venda seja a prazo e o cliente estiver com contas a receber em aberto acima do seu limite de crédito, o sistema irá impedir que ela seja efetuada. Se a venda for feita a vista, ela será concluída.

Figura 2: Fatura de cliente inativo.


Podemos verificar o valor do limite de crédito do cliente em seu cadastro.

Figura 3: Limite de crédito no cadastro de clientes.


Para verificar qual o valor em aberto nas contas a receber desse cliente, basta acessarmos o módulo 'FIN0001 - Contas a receber' em <Financeiro\Contas a receber>.

Figura 4: Valor em Contas a receber do cliente.
