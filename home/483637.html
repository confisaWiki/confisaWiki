Avisar contas vencidas de clientes - Confisa Sistemas

Neste artigo veremos como configurar o Confisa ERP para Avisar contas vencidas de clientes no Faturamento.


Esta funcionalidade é bastante utilizada pelos comércios, para avisar os vendedores sobre contas vencidas em aberto, para o cliente no momento do faturamento, evitando assim de fazer novas vendas a prazo para o cliente devedor.

Para aparecer a mensagem e avisar quando tem contas vencidas dos clientes ao fazer o faturamento com condição de pagamento a prazo, deve alterar para "SIM" a configuração 'Avisar contas vencidas do cliente no pedido (exceto para condição a vista)\Bloquear se contratado'  no 'CAD0019 - Cadastro de filial' na aba Configurações e salvar conforme demonstra abaixo:

Figura 1: Tela do cadastro de filiais.


Depois de configurado o passo acima no momento de salvar/efetivar o faturamento irá aparecer a mensagem que o cliente tem contas vencidas e se deseja prosseguir com o faturamento conforme demonstra abaixo:

Figura 2: Tela do cadastro de faturamento.
