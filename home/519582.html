Configurar emissão de NFSe com API para GeisWeb - Confisa Sistemas

URL: https://agile.movidesk.com/kb/pt-br/article/519582

Neste artigo veremos como configurar o sistema para emitir NFS-e em servidores GeisWeb, quando ele utilizar a API do ACBr.



O sistema permite transmitir NFS-e para prefeituras utilizando a API do ACBr, a qual permite a transmissão de notas para quaisquer prefeituras que utilizem essa API, dispensando programações específicas para cada uma.


Para configurarmos o sistema, devemos configurar os campos de NFS-e contidos na filial, as quais podem ser acessadas através da aba 'Configurações Adicionais' e subaba 'Diversas'. Para localizar as configurações, basta clicar no campo 'Descrição' e digitar 'nfse', apertando F3 para filtrar.

Figura 1: Configurações de API para NFS-e.



- NFSEIBGEPROVEDOR: Essa configuração é que vai saber para qual provedor da prefeitura será transmitido a nota (GeisWeb, 4R, DeISS, etc).

- NFSETIPOLANCTO: Essa informação deve ser solicitada na prefeitura, ou terá informações no próprio manual de integração.

- NFSECODTRIBUTACAO: Essa informação deve ser solicitada na prefeitura, ou terá informações no próprio manual de integração.

- NFSETIPOOPTANTESN: Informar o tipo de optante.

- NFSEVERSAOPROVEDOR: Nem sempre é necessário, apenas se tiver uma versão em especifico.

- NFSESITUACAOTRIB: Nem sempre é necessário, apenas se tiver no layout.

- NFSEAPIACBR: Deixar como 'Sim' para enviar NFS-e para prefeitura através da API do ACBR


Caso a empresa seja Regime normal,  é preciso escolher a opção 'Outros' para emissão de NFS-e em GeisWeb.


Após feitas as alterações necessárias no sistema, deve-se fechar e iniciá-lo novamente.


DEMAIS INFORMAÇÕES:

- Ter o certificado digital tipo A1 (.pfx) informado na filial;

- Informar a Inscrição Municipal da empresa na aba ;

Figura 2: Inscrição Municipal.



- O código do serviço da nota fiscal que é mandado no layout é o Código da ISSQN da nota. Para que  sistema transmita a nota e receba autorização, é preciso que seja informado o ISSQN aceito pela prefeitura. Essa informação você deve ser pega direto com a prefeitura ou contador, pois cada uma pode ter um código especifico.
