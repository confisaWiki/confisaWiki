Mostraremos neste artigo como configurar a integração entre o 'Confisa ERP' e sua plataforma de vendas online.


Abra o módulo pelo menu <Faturamentos\Vendas Online/Externas\Configurar Integração de Vendas Online>
Figura 1: Acessando o menu de configurações de integração de vendas online.


Clique no botão 'Adicionar', e preencha o campo 'Nome da integração'. No campo 'Ativo' informe 'Sim' e clique na lupa ao lado do campo 'Cód. Módulo' para selecionar uma integração, como a 'SIS0124 - Intergração Confisa Marketplace':

Figura 2: Iniciando uma nova configuração.


Os campos 'Refresh Token' e 'Token' são fornecidos pelo integrador do marketplace e pode-se obter novos tokens automaticamente clicando no botão 'Renovar Token'. Para fazer um teste de conexão, clique sobre o botão 'Testar conexão'.


O campo 'Url' também é obtido junto ao marketplace contratado e cada plataforma utiliza uma URL padrão.


É possível adicionar um novo canal (a exemplo: Amazon, Riachuelo, Mercado Livre, Dafiti, entre outros) clicando no campo 'Adicionar Canal'. Para indicar o canal como ativo ou inativo, marque ou desmarque a caixa de seleção do campo 'Ativo':

Figura 3: Ativando e desativando um canal de vendas 


*Observações: 

 1)Para utilizar o canal Riachuelo, nas configurações de integração do site deve-se selecionar VTRINA como a plataforma integradora;

 2) Para utilizar o canal Magazine Luíza, em alguns casos pode ser necessário entrar em contato com o suporte da plataforma para serem habilitadas permissões para a integração;


Os campos 'Tab. preço promoção' e 'Repres' devem ser preenchidos caso exista uma tabela de preço promocional e representante exclusivo por canal (preenchimento não obrigatório):


Especificamente para o canal Mercado Livre, deve ser informado o id da loja no campo 'id da Loja' obrigatoriamente. Esta informação deve ser obtida junto da plataforma do Mercado Livre e preenchida conforme demonstra abaixo:

Figura 4: Tela para configurar vendas on-line.


Aba 'Configurações':

 

  • Cód. Pagto: Informe aqui qual será a forma de pagamento padrão para a integração, ela deverá ser cadastrada no módulo 'CAD0038 - Cadastro de Condições de Pagamentos'. Se tiver dúvidas sobre como cadastrar uma condição de pagamento, clique aqui.

  • Estoque: Determina em qual estoque o sistema irá baixar os produtos quando fizer um faturamento proveniente de venda online. É obrigatório o preenchimento do campo 'Estoque' para atender as vendas on-line.

  • 'Representante': Insira neste campo o representante que será responsável pelas vendas do site, gerando suas comissões conforme configurado no cadastro de representantes.

  • 'Tab. Preço': Caso este campo seja preenchido, o sistema irá enviar ao site os preços de venda conforme a tabela de preço indicada. Se não for informada nenhuma tabela de preço, o sistema enviará os preços conforme o cadastro de cada produto.

    'Tab. Promoção': Este campo permite enviar ao site uma tabela de preços específica para promoções, diferente da tabela de preços normal.


Figura 5: Tela para configurar vendas on-line.
