Neste artigo veremos como enviar fotos de produtos para o Confisa Mobile, onde o representante poderá mostrar ao seu cliente o produto que estará comprando.



Primeiramente é preciso seguir algumas regras na organização das fotos. Para serem visualizados no Confisa Mobile devem estar no seguinte layout:


Formato de nome das fotos

códigodoproduto.jpg

00000001.jpg


Para casos onde o cliente usa cor:

códigodoproduto_codcor.jpg 

022E_BRANCO.jpg


É importante frisar que a pasta com as fotos deve ter no máximo 100 fotos quando for clicado em 'Enviar'. Caso haja mais fotos para enviar, pode-se dividir elas em pastas diferentes, selecionando uma por vez para fazer o envio corretamente.

Figura 1: Enviando fotos.



Obs: As extensões devem ser em JPG ou PNG e a resolução deve ser de no máximo 800x600 com 2 mb por foto.
