Cadastrar Código de Barras no Produto - Confisa Sistemas

Neste artigo veremos como cadastrar códigos de barras nos produtos.


1) Para códigos de barras EAN-13:

1.1 - Acesse o menu da filial pelo caminho <Configurações\Cadastro de filiais/empresas>. Em seguida, clique duas vezes na filial que deseja aplicar a configuração.

1.2 - Selecione a aba 'Configurações adicionais' e 'Etiquetas'. Depois, acesse a aba 'EAN13'. Clique no botão 'Incluir', indique o prefixo, o código atual e o limite máximo de etiquetas:

Figura 1: Cadastro de Filiais.


1.3 - Em seguida, acesse o cadastro do produto que deseja incluir o novo código de barras. Acesse a aba 'Código de Barras' e clique no botão 'Gerar códigos de barras'. Caso seus produtos utilizem Cor e Tamanho, será gerado um código de barra para cada cor e tamanho:

Figura 2: Inserindo código de barras para cada cor e tamanho.


1.4 - Para a exportação dos códigos de barras ao GS1, é necessário indicar o Peso Bruto e Peso Líquido dos produtos. Acesse a aba 'Classificações' e indique os pesos nos campos abaixo:

Figura 3: Cadastro de Produtos.


Observação:  Quando o código é do GS1 ele é enviado para nota de transferência, códigos que não sejam do GS1 não serão enviados.


2) Para códigos de barras que não são EAN-13

Acesse o cadastro do produto e selecione a aba 'Código de Barras'. Em seguida clique no botão 'Adicionar' e selecione um tamanho. Depois, clique sobre ícone de raio 'Gerar código de barras sequencial', conforme imagem abaixo:

Figura 4: Cadastro de Produtos.


Além do código sequencial, é possível indicar manualmente o código de barras clicando no botão 'Adicionar':

Figura 5: Inserindo código de barras manualmente.


Se o código de barras inicia com o número '2', é possível ajustar a configuração que valida o código de barras para balança. Acesse novamente o menu de configurações da filial e siga até a aba 'Configurações Adicionais'. Clique na coluna 'Descrição', digite 'balança' e pressione 'F3', como demonstra a figura abaixo:

Figura 6: acessando as configurações da filial.


Em seguida, indique no campo 'Valor' a opção 'Não'. Dessa forma você poderá cadastrar códigos de barras que iniciam com o número 2, mesmo que não utilize balança.


Se for exibida uma mensagem indicando que o código de barras já está sendo usado por outro produto, é possível removê-lo e adicioná-lo no produto desejado:

Figura 7: Removendo o código de barras e indicando o mesmo em outro produto.


*Nota: Para produtos comprados, o código de barras é cadastrado automaticamente conforme indicado no XML do fornecedor.
