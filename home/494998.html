Integração com e-commerce (Visual) - Confisa Sistemas

Veja como utilizar o módulo de integração, o qual permitirá sincronizar informações entre o Confisa ERP e seu site de e-commerce utilizando a 'Visual' para efetuar vendas online.

 

 

Pré-requisitos:
a) Antes de utilizar esse módulo é preciso fazer a configuração dele. Clique aqui para ver os detalhes.
b) Os produtos que serão vendidos online devem estar previamente cadastrados e configurados para que sejam exportados ao site. Clique aqui para ver detalhes sobre cadastro de produtos.

 

Depois que a configuração da integração estiver concluída e os produtos cadastrados e configurados, entre no módulo de Integração pelo menu <Faturamentos\Vendas Online / Externas\Integração de vendas online>. ou aperte 'CTRL + F' para abrir a janela de pesquisa e procure por 'SIS0090 - Integração de vendas online'.

 
Figura 1: Acessando o módulo de Integração de vendas online.


 

NOTA:


 - Importar: Trazer informações do site de e-commerce para o Confisa ERP.
          Exemplo: Cadastrar no sistema um pedido que foi feito no site.

 - Exportar: Enviar informações do Confisa ERP para o site de e-commerce.
          Exemplo: Disponibilizar no site um produto que já está cadastrado no sistema.

 

 

Veja abaixo como usar o módulo de integração de vendas online.

  

Exportar produtos:

Clique nas abas 'Produtos' 

 

'Importar Produtos'
Caso já use o Virtual e tenha produtos no site que precisa vincular com o sistema é possível fazer a importação dos produtos (esse processo só é necessário fazer uma vez, depois só será feita a exportação).  O campo que o sistema usa para vincular é o SKU, esse campo fica no Virtual e ele que será relacionado para vincular o código do produto no sistema. Clique em 'Listar do site' para puxar os produtos, 'selecione' o produto que deseja 'cadastrar' e clique em 'atualizar no sistema', como no exemplo a seguir:

Figura 2: Aba importar produtos.

 

 

'Exportar Produtos'.

Nesta aba é possível exportar produtos que estão cadastrados no sistema, enviando-os ao site com seus preços atualizados, NCMs e outros dados que são mostrados na tela.

Para realizar a exportação, clique no botão 'Listar do Sistema', selecione os produtos que deverão ser exportados ao site e clique no botão 'Atualizar no Site'. 

Figura 3: Exportar Produtos.

 

 
Importar pedidos:

Clique na aba 'Pedidos'.

Para importar os pedidos feitos no site, clique no botão 'Listar do Site', selecione quais pedidos deverão ser importados para o sistema, e clique no botão 'Atualizar no Sistema'.
O sistema mostrará quais pedidos já estão cadastrados no sistema com a cor verde, e os pedidos ainda não cadastrados na cor vermelha.


Para pesquisar pedidos lançados dentro de um período específico, preencha o campo 'Data do pedido'. O sistema mostrará os pedidos que foram cadastrados dentro do período informado. 

Figura 4: Importar Lista de Pedidos.


Para que o sistema relacione um pedido originado do marketplace com o representante e indique corretamente o intermediador na nota fiscal emitida dos pedidos, é necessário configurar a integração para usar marketplace. Para isso, acesse o menu <Faturamentos\Vendas Online/Externas\Configuração integração vendas online> e pesquise pela integração, nesse caso a Visual E-Commerce. Procure pelo campo 'Marketplace', e altere o valor para 'Sim':

Figura 5: Alterando a configuração 'Marketplace' nas configurações da integração.


Dessa forma, quando for cadastrado pedidos de marketplace (do Mercado Livre, por exemplo), o representante da venda será a entidade Mercado Livre. Dessa maneira, o intermediador marketplace na nota fiscal será o Mercado Livre.


*Observação: É necessário que o Nome/Razão Social do representante esteja cadastrado exatamente como 'Mercado Livre':
