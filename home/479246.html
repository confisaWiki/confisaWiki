Neste artigo veremos como cadastrar estoques e como indicar o seu código contábil, campo obrigatório para geração dos arquivos para o SPED.


Para cadastrar um estoque, acesse o menu <Estoque\Complementares\Cadastro de Estoque> como abaixo:

Figura 1: Acessando o menu de Cadastro de Estoque.


Observe que já existe um estoque cadastrado por padrão, esse é o estoque principal utilizado pela filial. Para cadastrar um novo estoque, clique em 'Adicionar' e indique um nome para o registro:

Figura 2: Cadastrando um novo estoque e indicando o código contábil.


Dessa maneira será possível gerenciar no sistema um segundo estoque para entradas e saídas de produtos e matérias-primas (Recurso frequentemente utilizado para vendas em marketplace, por exemplo).

Aponte a câmera do seu dispositivo Android para o QR code acima, ou clique aqui.
