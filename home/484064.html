Configurando filial com marketplace - Confisa Sistemas

URL: http://192.168.0.1:3000/pt-br/home/484064.html

Neste artigo veremos como configurar uma filial do sistema para utilizar o markeplace.


Para que o sistema possa trabalhar com o marketplace corretamente, é preciso também fazer algumas configurações na filial, onde os produtos serão enviados e os pedidos serão recebidos do site.

Primeiramente devemos configurar a conta corrente onde os valores recebidos serão lançados.

Devemos para isso ir no módulo 'CAD0019 - Cadastro de Filiais' através da combinação 'CTRL + F' e digitar 'filiais', dando 'Enter' a seguir.

Já no módulo citado acima, pesquisarmos a filial e iremos à aba 'Financeiro', onde iremos indicar a conta corrente onde os valores recebidos no marketplace serão lançados no campo 'Conta da quitação de faturas do Confisa marketplace'. Essa conta pode ser uma já existente ou uma nova desejada, a qual pode ser cadastrada através do módulo 'CAD0041 - Cadastro de Contas'. Caso tenha dúvidas sobre como cadastrar contas, clique aqui.

Figura 1: Cadastro de Filiais.


Feita a indicação da conta corrente, iremos também indicar um usuário o qual fará a baixa dos pedidos automaticamente, além de renovar o token a cada 20 horas. Para isso, iremos na aba 'Sistema' e indicaremos o usuário no campo 'Login da sincronização automática do Confisa marketplace'.

É altamente recomendado que esse usuário seja cadastrado exclusivamente para isso, não devendo ser um usuário utilizado por uma pessoa física.

Figura 2: Usuário de sincronização automática.

No caso de o usuário Não selecionar na filial uma conta corrente na aba 'Financeiro' e no campo 'Conta da quitação de faturas do Confisa marketplace', será solicitada a conta para quitação de cada pedido conforme eles forem sendo recebidos. 

Este caso serve para o usuário que recebe os pedidos em contas correntes diferentes uns dos outros.


Obs: É importante salientar que esse usuário será utilizado apenas pelo sistema, o qual fará uso de um serviço interno dele para realizar essa sincronização dos pedidos.
