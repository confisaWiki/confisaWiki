Veja como utilizar o módulo de integração, o qual permitirá sincronizar informações entre o Confisa ERP e seu site de e-commerce utilizando a 'Loja Integrada' para efetuar vendas online.

 

Pré-requisitos:
a) Antes de utilizar esse módulo é preciso fazer a configuração dele. Clique aqui para ver os detalhes.
b) Os produtos que serão vendidos online devem estar previamente cadastrados e configurados para que sejam exportados ao site. Clique aqui para ver detalhes sobre cadastro de produtos.

 

Depois que a configuração da integração estiver concluída e os produtos cadastrados e configurados, entre no módulo de Integração pelo menu <Faturamentos\Vendas Online/Externas\Integração de vendas online (Manual)>.

 

Figura 1: Acessando o módulo de integração de vendas.


NOTA:


 - Importar = trazer informações do site de e-commerce para o Confisa ERP.
          Exemplo: Cadastrar no sistema um pedido que foi feito no site.

 - Exportar = enviar informações do Confisa ERP para o site de e-commerce.
          Exemplo: Disponibilizar no site um produto que já está cadastrado no sistema.

 

 

Veja abaixo como usar o módulo de integração de vendas online.

 

1) Importar clientes do site:

Clique na aba 'Clientes'.

Nesta aba é possível importar os clientes que estão cadastrados no site.
Clique no botão 'Listar do Site' para visualizar os clientes que se cadastraram no site. Marque a caixa à esquerda dos clientes que deverão ser importados e em seguida clique no botão 'Atualizar no Sistema'.


NOTA: Ao proceder conforme descrito acima, todos os clientes cadastrados no site serão listados, independente de terem efetuado compras ou não.

Figura 1: Importar Clientes.

  

 

2) Exportar e Importar categorias:

Clique nas abas 'Categorias' e 'Exportar Categorias'.


Categorias são utilizadas para separar e organizar os produtos no site através de menus, onde cada categoria é um menu.

Para exportar categorias dos produtos, clique no botão 'Listar do Sistema'. Em seguida, marque as categorias que deverão ser exportadas para o site a partir do sistema, e clique no botão 'Atualizar no Site'. 

Figura 2: Exportar Categorias.

  

 

3) Exportar Marcas:

Clique nas abas 'Marcas (Famílias)' e 'Exportar Marcas'.

As marcas de um produto são cadastradas como 'Famílias' no sistema, e indicadas no Cadastro de produtos.

Para exportar uma marca, clique no botão 'Listar do Sistema', selecione as marcas a serem exportadas ao site, e clique no botão 'Atualizar no Site'.

Importante: A Loja Integrada obriga o uso de Marcas, por isso, é importante exportar as Marcas (Famílias) do sistema para o site antes do envio de produtos.

Obs: Somente será exibida a aba 'Marcas' caso indicado na configuração da integração a opção 'Usa marcas' - Sim. 

Figura 3: Exportar Marcas.

 

 

4) Exportar grades de Cor e Tamanho:

Clique na aba 'Cores e Tamanhos (Grade)'.

Esta aba permite visualizar e exportar as cores e tamanhos que estão cadastrados no sistema.

Para listar e exportar as cores e tamanhos cadastrados no sistema, clique no botão 'Listar do Sistema', selecione os tamanhos a serem exportados e clique no botão 'Atualizar no Site'.

Figura 4: Exportar Cores e Tamanhos.

  

 

5) Exportar produtos:

Clique nas abas 'Produtos' e 'Exportar Produtos'.

Nesta aba é possível exportar produtos que estão cadastrados no sistema, enviando-os ao site com seus preços atualizados, NCMs e outros dados que são mostrados na tela.

Para realizar a exportação, clique no botão 'Listar do Sistema', selecione os produtos que deverão ser exportados ao site e clique no botão 'Atualizar no Site'.

Figura 5: Exportar Produtos.

 

 

6) Importar pedidos:

Clique na aba 'Pedidos'.

Para importar os pedidos feitos no site, clique no botão 'Listar do Site', selecione quais pedidos deverão ser importados para o sistema, e clique no botão 'Atualizar no Sistema'.
O sistema mostrará quais pedidos já estão cadastrados no sistema com a cor verde, e os pedidos ainda não cadastrados na cor vermelha.

Para pesquisar pedidos lançados dentro de um período específico, preencha o campo 'Qtde Dias a Pesquisar'. O sistema mostrará os pedidos que foram cadastrados dentro do período informado, considerando a data de lançamento do pedido que consta no campo 'Data Pedido'.

Figura 6: Importar Pedidos.
