Veja neste artigo como configurar a integração entre o Confisa ERP e um site com Visual.

 

 

Abra o módulo pelo menu <Faturamentos\Vendas Online / Externas\Configurar Integração de Vendas Online> ou aperte 'CTRL + F' para abrir a pesquisa e procure por 'SIS0089 - Configurar Integração de Vendas Online'.


Figura 1: Acessando o módulo.


Clique no botão 'Adicionar', e preencha o campo 'Nome da integração' conforme o site que você contratou. Marque o campo 'Ativo' como 'Sim'.

Clique na lupa ao lado do campo 'Cód. Módulo' para selecionar a integração 'Visual' contratada.

Informe a 'Url' do site e o 'Token' (quem passa essa informação é o suporte do site que foi contratado).

 

Aba 'Configurações':
 

  • 'Estoque': Determina em qual estoque o sistema irá baixar os produtos quando fizer um faturamento proveniente de venda online.
  • 'Representante': Insira neste campo o representante que será responsável pelas vendas do site, gerando suas comissões conforme configurado no cadastro de representantes.
  • 'Tab. Preço': Caso este campo seja preenchido, o sistema irá enviar ao site os preços de venda conforme a tabela de preço indicada. Se não for informada nenhuma tabela de preço, o sistema enviará os preços conforme o cadastro de cada produto.
  • 'Promoção': Este campo permite enviar ao site uma lista de preços específica para promoções, diferente da tabela de preços padrão.

Figura 2: Configurações.


Preencha todos os campos com os dados informados pelo site, conforme exemplo da 'Figura 2' acima. Para testar a conexão, clique no botão 'Desconectado', dessa forma o sistema tentará fazer a comunicação com o site, alterando o botão para 'Conectado', indicando que o procedimento foi bem sucedido.


 

Na aba 'Status do pedido' é possível definir o status que tem no site que vai manter no sistema, esse status quem fornece é o site então tem que pegar os campos 'Descrição do status' e 'ID'   do site (quem passa essa informação é o pessoal de configuração do site).
