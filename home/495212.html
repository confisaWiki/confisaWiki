Conferência de Pedidos para Fatura quando usa lotes

Atualizado por Diego Cardoso em 23/09/2024 16:47

Neste artigo veremos como executar a conferência de pedidos para fatura, quando utiliza lotes no sistema.


O procedimento é semelhante à conferência de pedidos, a diferença é que deve-se indicar o lote dos itens quando existir. Para saber como realizar a conferência de produtos para fatura quando não usa lote, clique aqui.


Com o pedido de venda cadastrado, acesse o menu de conferência pelo caminho <Faturamentos\FAT0031 - Conferência de Pedidos para Fatura> ou digite as teclas de atalho 'CTRL + F' e procure pelo termo de busca 'conferência' ou o código do menu , 'FAT0031'. Clique no botão 'Pesquisa' para buscar pelo pedido. Veja na imagem abaixo:

Figura 1: Acessando o menu de conferência de pedidos.


Após selecionar o pedido a ser conferido, indique a quantidade a faturar. Pode ser utilizado o código de barras ou indicando manualmente a quantidade de cada item. Em seguida, procure pelo campo 'Tem lote' e dê um duplo clique nesse campo para acessar o menu 'Controle de lotes' do item:

Figura 2: Acessando o menu de controle de lotes na conferência de pedidos.


Observe que ao acessar o menu de controle de lotes. o sistema já traz pré-selecionado os lotes do item que possuem saldo disponível, totalizando a quantidade necessária para a conferência do pedido. Caso necessário, é possível alterar a quantidade no campo 'Qtde. a considerar' e também os lotes a serem considerados:

Vale notar que por padrão, o sistema pré-seleciona os lotes automaticamente conforme a ordenação do código do lote. Porém, é possível configurar a ordenação padrão para considerar o lote com a data de validade mais recente, por exemplo. Para mais detalhes, entre em contato com a equipe de suporte.


Caso um lote tenha sido relacionado por engano, clique no botão 'Desvincular Lotes', para recomeçar o procedimento:

Figura 3: Desvinculando lotes do item.


Caso necessário, é possível realizar a conferência parcial de um pedido. Para isso, insira a quantidade a ser faturada no campo 'Qtde. Faturar' e no controle de lotes indique a quantidade a ser faturada, no campo 'Qtde a considerar'. Veja o procedimento na imagem abaixo:


Se necessária a conferência total do pedido, preencha o campo 'Qtde a Faturar' de forma que o valor seja a quantidade total do pedido e no controle de lotes, indique a quantidade no campo 'Qtde a Considerar':

Figura 4: Conferência total do pedido quando usa lotes.


Ao finalizar a vinculação dos lotes, finalize a conferência clicando no botão 'Conferir Total'.
