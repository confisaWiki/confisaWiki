Leitura do arquivo de remessa bancária 



Abra o arquivo de remessa com o bloco de notas e para facilitar, verifique se a quebra de linhas está desativada.

Com a cartilha do banco e leiaute correto (pode ser 240, 400 ou outros), abra a cartilha e verifique o HEADER (informações na primeira linha).

Conforme for descendo as linhas do arquivo de remessa, o local da cartilha onde estão as informações com as legendas sobre os campos vai alterar.


Legenda dos campos:

LN= LINHA

COL= COLUNA

Figura 1: Leitura de arquivo de remessa.
 


IMPORTANTE:

Cada banco pode possuir nomenclaturas diferentes para o mesmo campo ou informações diferentes. Usem o arquivo que pegamos as informações dos campos para os clientes passarem para os bancos para saber se esse banco usa esse campo de alguma outra forma.
