Rejeição 612: GTIN da unidade tributável (cEANTrib) inválido - Confisa Sistemas

Neste artigo veremos como resolver a rejeição 612 ao emitir uma nota fiscal.


Essa rejeição diz respeito à informação do código de barras dos produtos na nota fiscal. Se a nota foi digitada manualmente, verifique se os campos EAN e EAN13 estão preenchidos conforme as regras abaixo:

Se os produtos utilizam o código de barras EAN13:
-> Código de 13 números


Caso não utilizem o código de barras EAN13:
->Termo literal: "SEM GTIN" 


Na nota fiscal:

Figura 1: Produto sem código EAN que não será validado.


Nos casos onde o produto não terá código de barras, devemos informar "SEM GTIN". Se a nota foi iniciada sem que o produto tivesse um código de barras EAN, o sistema irá colocar essa informação automaticamente.

Figura 2: Produto com informação válida.
