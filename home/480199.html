Cadastro, Liberação e bloqueio de módulos e permissões para usuários

Atualizado por Diego Cardoso em 22/07/2024 11:01

Neste artigo veremos como criar um novo usuário, configurá-lo, liberar e retirar módulos, liberar todos os módulos, apagar sua senha e copiar os acessos de um usuário para outro.


Para acessar o cadastro de usuários, devemos clicar no menu <Configurações\Cadastro de usuários do sistema>. A seguir, para cadastrar um novo usuário, basta clicar em 'Adicionar'.

Figura 1: Acessando o módulo de 'Cadastro de Usuários do Sistema'.


1 - Criar um novo usuário
Tendo iniciado um novo usuário, nos campos 'Login' e 'Nome de Usuário' qual será o login a ser utilizado. Feito isso, basta salvar para que o sistema já coloque os caixas a qual ele terá acesso.

Obs: Mesmo que o usuário não vá trabalhar com o financeiro, recebimento ou pagamento de contas, o sistema indicará os caixas como citado.


Figura 2: Criar um novo usuário.


2 - Liberar e Retirar Módulos.

Para liberar e retirar módulos, é preciso ir à aba 'Módulos' e selecionar o tipo de módulo a ser liberado. Clicamos em '+' no botão logo abaixo, dessa forma o sistema mostrará a lista de módulos disponíveis no sistema para que sejam escolhidos.
É possível notar que o sistema permite que mais de um módulo seja adicionado, bastando marcar a caixa do lado esquerdo dele, tornando possível assim fazer o processo de liberação mais rapidamente.
Caso deseje retirar algum módulo, basta clicar sobre o desejado e a seguir em 'Excluir' conforme mostra a figura 2.


Figura 3: Liberação de Módulos.


3 - Liberar todos os módulos e limpar senha.

O sistema permite também que todos os módulos sejam liberados de uma vez, para isso basta clicar no botão 'Liberar Tudo', o qual tem o ícone de um cadeado, assim o sistema perguntará se deseja-se liberar todos os módulos, bastando clicar em 'Sim'.


Figura 4: Liberar todos os módulos e limpar senha.


4 - Copiar acessos de um usuário para outro ou criar um novo usuário com os mesmos acessos e limpar senhas.

Para limpar a senha de um usuário, caso tenha-se esquecido qual é, basta clicar no botão 'Limpar senha cadastrada'.

Figura 5: Limpar senha cadastrada.


Ainda é possível no sistema copiar os acessos de um usuário para outro, ou ainda criar um novo usuário com os acessos do usuário aberto no momento. Para isso, basta acessar o usuário que possui os acessos desejados, o qual servirá como base para o novo usuário ou definição de acessos para outro, clicar no botão 'Copiar e Colar' e digitar o 'Login' e 'Nome' do usuário', clicando em 'Sim' a seguir.

Figura 5: Copiar e Colar usuários.


5 - Definir conta padrão

É possível definir no usuário para ele visualizar só as contas relacionadas a ele deixado a caixa de seleção  'Visualizar\ Movimentar contas relacionadas', também é possível definir uma conta padrão tanto caixa como bancaria no usuário, assim quando for fazer um recebimento ele já vai trazer a conta informada.

Figura 6: Conta Padrão.


6 - Acesso a múltiplos terminais

Caso necessário, pode ser definido para um usuário acessar vários terminais ao mesmo tempo. Para isso, basta marcar a opção 'Permitir múltiplos acessos' como no exemplo a seguir:

Figura 7: Permitir múltiplos acessos
