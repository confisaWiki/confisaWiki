Erro ao iniciar MDF-e - Confisa Sistemas

Neste artigo veremos como resolver o erro ao iniciar um novo MDF-e. Normalmente o erro é causado pois falta informar para o sistema o número de série padrão do documento.


Obs: Caso não saiba qual é o número de série que se deve usar, consulte o escritório de contabilidade.


Para iniciar um novo MDF-e, acesse o módulo pelo caminho <Nota fiscal\Manifesto eletrônico MDF-e> ou pelas teclas de atalho 'CTRL + F' e pesquisando pelo termo de busca 'mdf-e':

Figura 1: Acessando o módulo de emissão de MDF-e


Observe que é gerado um erro ao incluir um novo documento. Para resolver, acesse as configurações da filial pelo caminho <Configurações\Filiais/Empresas>. Em seguida, seleciona a filial e selecione a aba 'Configurações Adicionais\Nota Fiscal\Configurações': 

No campo 'Série do MDF-e', indique o número de série padrão que será usado na emissão dos documentos:

Figura 2: Acessando as configurações de nota fiscal da filial


Caso não saiba qual é o número de série que deve usar, consulte seu escritório de contabilidade.


Salve as alterações. Observe que poderá iniciar normalmente um novo MDF-e:

Figura 3: Cadastrando um novo MDF-e.
