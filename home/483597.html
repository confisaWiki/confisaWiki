Neste artigo mostraremos como realizar o cadastro de item simplificado.


Para cadastrar um item simplificado, basta ir em <Cadastros\Itens/Produtos\Cadastro de item simplificado>.

Figura 1: Acessando o módulo.


Para cadastrar produtos, insira as informações (Descrição, Unidade, Preço Custo, Código de Barra, Etc) conforme exemplo abaixo:

Feito isso, o item já estará cadastrado no sistema.

Observação: No campo 'Adicionar Barra' é possível usar o leitor de código de barra para preenchimento automático. 


Pesquisa de itens
Para pesquisar produtos, selecione o tipo de item e marque na área 'Data de cadastro' o período de cadastro do produto, dessa forma é possível filtrar os itens que já estão cadastrados no sistema. 

Observação: para filtrar todos os itens já cadastrados, basta informar um período no qual possa mostrar até mesmo o primeiro item do sistema.
Exemplo: um período de dois anos (10/07/2024 até 25/07/2024).
 
Figura 2: Cadastrando produto.


Na aba 'Filtros Digitáveis' é possível filtrar o produto por código de barra ou por descrição.
 
Figura 3: Pesquisa de produtos.


Alterar informações dos itens

Com os itens na tela, é possível alterar as informações deles apenas clicando no campo correspondente (Descrição, Unidade, Ncm, etc). Assim, a informação alterada já estará constando no cadastro do item.

Obs: Quando o tipo do item for 'Produto', o sistema mostrará apenas NCMs que estejam marcadas como 'Produtos'. Quando o tipo for 'Serviço', apenas NCMs configuradas com o tipo 'Serviço'.
 
Figura 4: Editar itens.


Cadastrar itens por XML

Nesse módulo, é possível cadastrar também os itens que não estão no seu sistema através do XML da nota fiscal de recebimento.
Para realizar esse procedimento clique no botão 'Ações' e escolha a opção 'Cadastrar o item a partir de um XML de nota fiscal' e selecione o arquivo no seu computador.

Figura 5: Cadastro por XML.


Feito isso, os itens já estarão cadastrados no sistema.
