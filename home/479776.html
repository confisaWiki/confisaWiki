Romaneio de saída para oficinas - Confisa Sistemas

URL: http://192.168.0.1:3000/pt-br/home/479776.html

Neste artigo veremos como gerar um romaneio de saída para oficinas, internas ou externas.


Primeiramente, acesse a ordem de produção pelo caminho <Produção\Ordem de produção>. Para saber mais sobre ordens de produção no Confisa ERP, clique aqui. É necessário indicar que a oficina pode executar o processo de produção que se deseja gerar o romaneio. Para saber como realizar esse procedimento, clique aqui.


Selecione o processo para o qual será gerado o romaneio e clique no botão 'Ações.'  e em seguida 'Gerar romaneios de oficinas para o processo posicionado'. Observe a imagem abaixo:

Figura 1: Gerando o romaneio de saída para um processo na ordem de produção.


Na tela que se abre clique no botão 'Escolher Oficinas', para selecionar a(s) oficina(s) que irão executar o processo. Para um único processo, é possível selecionar mais de uma oficina e também selecionar condições de pagamento ou valores distintos para cada uma. Observe um exemplo na imagem abaixo:

Figura 2: Gerando romaneio de oficinas de um único processo para duas oficinas.


Ao clicar no botão 'Gerar', será exibido uma mensagem perguntando se deseja imprimir os romaneios, ou vales, para cada oficina:

Figura 3: Impressão do romaneio.


Pronto! O romaneio já está gerado e vinculado à OP. O romaneio pode ser acessado clicando na aba 'Romaneio para Oficinas', como mostra a imagem abaixo:

Figura 4: Acessando o romaneio gerado pela OP.


Para saber como dar entrada nos itens que a oficina retornar, clique aqui.


Caso ao tentar gerar o romaneio não seja possível escolher a oficina e apareça o aviso informando que não está informando a quantidade, mesmo com ela estando sendo informada, como exemplo a seguir:

Figura 5: Alerta da quantidade.


Verifique se no 'Processo de Produção' (PRD0004) está informado na aba 'Oficina' alguma oficina, caso não esteja acrescente a oficina como no exemplo abaixo:

Figura 6: Acerto no 'PRD0004 - Cadastro de Processos de Produção'.


Caso necessário é possível atualizar a quantidade enviada caso decida enviar outra parte em outro romaneio ou até não produzir tudo, basta no romaneio editar a quantidade e imprimir novamente o romaneio, como no exemplo a seguir:

Figura 7: Edição do romaneio.


Nota: Lembrando que se for diminuir mesmo a quantidade em produção é necessária acertar na OP a quantidade correta.
