Nota Fiscal de Produtos e Serviços - Confisa Sistemas

Neste artigo veremos como emitir notas fiscais de serviço e produto a partir de uma fatura, de modo que o sistema separe os dois automaticamente para posterior transmissão.


O sistema Confisa ERP permite que seja feita uma fatura com produtos e serviços nela, separando os dois tipos posteriormente em duas notas separadas, dessa forma o usuário pode fazer a transmissão de cada uma no seu devido lugar, visto que para muitas empresas deve-se transmitir a nota de serviço para a prefeitura de sua cidade.

Para que o sistema faça esse trabalho, devemos ir no menu <Configurações\Cadastro de Filiais/Empresas>. A seguir iremos na aba 'Configurações Adicionais' e clicamos em 'Descrição', procuramos também a configuração 'Separar Produtos e Serviços na Nota Fiscal', marcando-a como 'Sim'.

É possível ver ainda que o sistema gerará duas contas a receber, uma para a nota com os produtos e outra apenas para a nota com os serviços, dessa forma poderá ser feito um controle financeiro melhor.
