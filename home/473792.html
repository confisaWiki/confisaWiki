Neste artigo veremos como cadastrar uma Família de produtos.


A família de produtos serve para podermos categorizar nossos produtos, melhorando a organização e nos permitindo obter relatórios utilizando os filtros por classificação de produtos. ( ex. relatório de vendas por família de produtos). 


Para acessar o módulo de cadastro de família, acesse o menu <Cadastro\Itens/Produtos\Especificações de Produtos\Famílias> ou digite as teclas de atalho 'CTRL + F' e procure pelo termo de busca 'Família':
Figura 1: Cadastro de Famílias


Clique no botão 'Inserir' para indicar uma nova classificação. É possível atribuir uma descrição e uma % de desconto ( não obrigatórios ) para a divisão:

Figura 2: Cadastro de Família


Para classificar os itens, acesse o cadastro do produto ou matéria-prima em questão e indique a família a qual o item pertence:

Figura 3: Cadastro de Produtos


Por fim, caso queira remover alguma família, é possível deletá-la do sistema clicando no botão 'Excluir'. 

Obs: Não é possível deletar uma família se ela estiver sendo utilizada para classificar algum item no sistema:

Figura 4: Cadastro de Família
