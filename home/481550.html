Cadastro de NCM - Confisa Sistemas

Nesse artigo iremos aprender a cadastrar manualmente uma NCM.


Por padrão a NCM já é cadastrada no sistema ao fazer a importação da Tabela IBPT, caso não se lembre do procedimento clique aqui.

Eventualmente o governo pode alterar a validade da NCM ou decidir mudar para uma nova, fazendo com que quando for vinculado o cadastro do produto à NCM, apareça o seguinte aviso: 'Classificação Fiscal NCM não cadastrada', como no exemplo a seguir:

Figura 1: NCM não cadastrado.


Verifique com seu escritório de contabilidade se a NCM está correta e válida, caso esteja e não conste no sistema para associar será preciso cadastrar manualmente, para isso acesse o menu <Cadastros\Operações Fiscais\NCM>, clique no botão 'Adicionar' para inserir uma nova NCM, no campo Código informe a NCM desejada, no campo Descrição informe a nomenclatura conforme o escritório passou e clique no botão 'Salvar'.

Figura 2: Cadastro de NCM.


Observação: o Cód. EX TIPI deve ser preenchido caso a NCM seja a mesma, mas contenha uma pequena variação. Por padrão, o campo precisa conter duas casas decimais, como o exemplo à frente: Se por exemplo for o código '1', preencher como '01'.

Em alguns casos ao importar o XML de entrada da NF-e ou mesmo se tentar selecionar um item para adicionar na NF-e manualmente, aparece a mensagem que o NCM não está cadastrado para determinado item.

Neste caso, deve-se verificar se o NCM tem o código EX TIPI ou não na mensagem exibida pelo sistema, caso esteja sem o código, faça o cadastro como demonstrado abaixo:

Figura 3: Tela do cadastro de NCM.


Por fim, com a NCM cadastrada e configurada, podemos fazer a troca no produto desejado.

Figura 4: Ajuste de cadastro de produto.
