Pedidos de venda com imagens - Confisa Sistemas

Neste artigo veremos como inserir imagens nos produtos que estão sendo vendidos.



Podemos inserir várias imagens nos produtos que estão informado no pedido de vendas, dando detalhes ao cliente sobre as características visuais dos mesmos. Para isso, basta clicar sobre um produto e clicar no botão 'Adicionar'. O sistema abrirá uma janela para seleção de imagens, onde poderemos escolher a imagem desejada.


Após selecionada a imagem, devemos colocar uma descrição da foto no pedido. É importante salientar que o modelo que traz as imagens no pedido é o modelo 25.

Figura 1: Selecionando imagens para os produtos.



Obs: Se não houver fotos do produto no pedido, o sistema irá buscar as fotos contidas no cadastro dos mesmos.



Para habilitar a impressão de imagens nos pedidos, devemos habilitar a configuração 'Salvar a imagem do cadastro de produto compactada'. Dessa forma o sistema habilitará os campos de imagens no módulo.
