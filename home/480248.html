Local onde está salvo o borderô de remessa

Este artigo tem por objetivo identificar e/ou alterar onde estão sendo salvos os arquivos de remessa de borderôs de cobrança bancária.


Para identificar e/ou alterar o caminho dos arquivos de remessa, abra o módulo 'Cadastro de Contas' pelo caminho <Cadastros/Operações Financeiras/Cadastro de Contas>, ou pressione as teclas de atalho 'CTRL+F' para abrir a tela de pesquisa de módulos e digite 'Contas' ou 'Cadastro de Contas'.


Indique o código da conta ou se preferir pesquise a conta bancária através da lupa. Em seguida, na aba 'Cobrança bancária', localizada na parte inferior da tela, procure pela coluna 'Caminho do arquivo de remessa'. Este é o local que o sistema irá utilizar para salvar os arquivos de remessa gerados. Para alterá-lo, basta clicar duas vezes sobre a célula ou clicar uma vez sobre o ícone dos três pontinhos, como na imagem abaixo:
Figura 1: Cadastro de Contas


Pronto! O caminho para salvar os arquivos de remessa que são enviados para o banco foi definido. Para saber como gerar os arquivos de remessa, clique aqui.


Os arquivos de remessa serão salvos no caminho configurado, como podemos ver no exemplo abaixo: 

Figura 2: Caminho do arquivo gerado.
