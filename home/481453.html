Exportação de dados de consulta - Confisa Sistemas

URL: https://agile.movidesk.com/kb/pt-br/article/481453

Neste artigo veremos como utilizar o módulo de exportação de dados de consulta:


O módulo se encontra em <Relatórios\Cadastros\Exportação dos dados de consulta>.

Figura 1: Acessando o módulo.


- Aba 'Totalizadores': 

Informe no campo a informação que será totalizada (podendo ser SOMA, MÉDIA, VALOR MÁXIMO OU VALOR MÍNIMO), conforme exemplo abaixo:

- Aba 'Condições':

Campo 'Parâmetro': Defina o parâmetro que será utilizado no SQL para que seja substituído pelo código SQL que você definir em 'Condição SQL'. O parâmetro deve estar entre chaves {EXEMPLO}, tanto na aba quanto no Script SQL e devem ser exatamente iguais nos dois lugares, conforme exemplo abaixo:

- No campo 'Condição SQL', deve-se informar o código SQL que será usado para fazer o filtro na consulta. Deve-se informar o símbolo '$', colocando no lugar da informação a ser filtrada para que o sistema substitua pelo filtro que o usuário deseja fazer através de uma janela que o sistema mostrará. No exemplo abaixo, serve para o usuário poder filtrar por um código de representante específico.

Defina o 'Tipo' (pode ser texto, número, data ou outros) e a 'Ordem' que irá aparecer na tela 'Minhas Consultas' que fica no caminho <Relatórios\Minhas Consultas>.

Defina a 'Função Pesquisa', caso o tipo do parâmetro seja de pesquisa. Ex: Se quiser pesquisar por representante, é necessário colocar a função 'Representante', basta selecionar entre as que já vem por padrão.   

- Aba 'Usuários': Usuários que terão permissão para tirar o relatório em questão.
