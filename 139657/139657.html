Configuração de Google Cloud e Chave API

Para configurar a conta do Google para uso do Google Cloud no Sistema Confisa ERP, deve-se primeiro acessar o site <https://cloud.google.com/>. Após isso é necessário configurar uma conta do Google conforme a figura mostra abaixo.

Figura 1: Acesso de Site.

 

 

Com a conta configurada, deve-se primeiramente criar a chave API, a qual será utilizada no sistema. Para isso basta clicar no botão <Credenciais\Criar nova chave API>.

Figura 2: Criação de Credenciais.

 

 

A seguir deve-se ativar duas bibliotecas necessárias para que o sistema possa acessar o mapa no Google e mostrar os dados das cidades onde o representante fez vendas no sistema. Para isso basta clicar em 'Bibliotecas' e digitar primeiramente 'Java', conforme mostra a figura, e clicar em 'Ativar'. Feito isso, deve-se também ativar o 'Geocoding', bastando clicar sobre o link contido na lista e clicar em 'Ativar'.

 

Figura 3: Ativação de Geocoding API.

 

 

Ao final deve-se clicar em 'Chave API' localizar a chave criada anteriormente e clicar no botão 'Copiar', dessa forma o código será copiado no sistema para que possa-se colocá-lo no sistema.

Figura 4: Cópia de Credenciais.

 

 

Após copiada a chave, deve-se abrir o Sistema 'Confisa ERP', clicar em <Sistemas\Filiais> e abrir a aba <Configurações>, digitar no campo 'Pesquisa' o texto "Chave" e apertar 'F3', clicar sobre o campo e apertar 'CTRL + V' para informar a chave copiada no sistema.

Feito isso basta salvar e fechar a tela, o sistema está configurado para usar o relatório de <Mapa de Cidades Vendidas por Representante>.

 

Figura 5: Informação de Chave API no Sistema.

 

 

 

Aponte a câmera do seu dispositivo Android para o QR code acima, ou clique aqui.
